# Microservices

Ця директорія містить всі мікросервіси проекту.

## Сервіси

- **auth-service** - Сервіс аутентифікації та авторизації
- **user-service** - Сервіс управління користувачами та друзями
- **project-service** - Сервіс управління проектами
- **task-service** - Сервіс управління завданнями

## Структура

```
services/
├── _template/          # Шаблон для створення нових сервісів
├── auth-service/       # Auth Service
├── user-service/       # User Service
├── project-service/    # Project Service
└── task-service/       # Task Service
```

## Документація

Детальна документація знаходиться в кореневій директорії проекту:
- `VERCEL_API_AUTH_SETUP.md` - Налаштування API authentication
- `API_KEYS_SETUP.md` - Налаштування API keys
- `SERVICES_ROADMAP.md` - Roadmap сервісів
